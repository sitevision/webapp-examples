# ai-assistant-recommendations

Example WebApp about how you can utilize the [aiAssistant SDK](https://developer.sitevision.se/docs/webapps/sdk/aiassistant). The WebApp is created as a complement to a presentation made for Sitevisiondagarna 2025.

Link to a recording of the presentation is coming soon.

The idea is to use RAG and AI to make suggestions for presentations to check out during the Sitevisiondagarna event. The user asks for a recommendation and gets a suggestion back. If you do not like the suggestions or a related presentation could not be found, give it another go with some other requests.

## Setup

In the site settings you need to set up a knowledge base with the desired scope. You will also need to set up an assistant. For the example prompt used. See aiassistant.txt.
